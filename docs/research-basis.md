# Research basis and interoperability targets

The proposal is informed by several complementary directions:

- Open X-Embodiment / RT-X: standardized multi-robot datasets and positive cross-robot transfer.
- RT-H: language-motion hierarchies as an interpretable intermediate layer between tasks and actions.
- UniAct: tokenized universal action spaces for cross-embodiment learning.
- FAST / FAST+: compression-oriented action tokenization for high-frequency action chunks.
- LeRobotDataset v3: scalable multimodal time-series, video, and metadata organization.
- ROS 2 Actions: goal, feedback, result, cancellation, and preemption for long-running robot behaviors.
- BehaviorTree.CPP: typed ports, blackboards, and composable behavior execution.
- GenSim and RoboGen: large-model-assisted task and simulation generation.
- MimicGen: scalable transformation of a small number of demonstrations into diverse datasets.
- RoboTwin 2.0: multimodal-model-assisted task-level code generation, simulation-in-the-loop refinement, domain randomization, and multi-embodiment evaluation.

OEBP is intended to connect these layers rather than replace them.
